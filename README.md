# NWebGame

Лёгкая multiplayer веб-игра в стиле "пожирай других": ешь еду, увеличивай массу и съедай меньших игроков.

## Почему тянет слабые устройства
- Canvas 2D без тяжёлых библиотек.
- Низкая частота сетевых апдейтов (input ~10 раз/сек, снапшоты 10 раз/сек).
- Передача только ближайших объектов.
- Плавная интерполяция на клиенте, чтобы скрывать лаги и потери пакетов.

## Запуск
```bash
npm install
npm start
```

Открой: `http://localhost:3000`

Для игры по сети открой порт `3000` на хосте и заходи по IP сервера.
